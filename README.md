# Responsividade

## CSS Units

Unidades de medidas do CSS

Layout Fixo
`px` - Pixels

Layout Fluido
`%` - Porcentagem
`auto` - Automática
`vh` - Viewport Height
`vw` - Viewport Width


Textos fixos
`1px` = 0.75pt
`16px` = 12pt

Textos fluídos
`em` - multiplicado pelo paia
`rem` - multiplicado pelo root/body/font-size

## Media Queries CSS

@media (max-width: 320px){
    #form h3{
        font-size: 2rem;
    }
}
